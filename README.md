# langchain-chatbot
Based on the Langchain framework, a retrieval and generative chatbot.

## usage
```
pip install -r requirements.txt
```

## copy env
```bash
cp .template.env .env
```

## edit env
edit your env config

## run chatbot
```
python chatbot.py
```

## run streamlit
```
streamlit run streamlit.py
```
